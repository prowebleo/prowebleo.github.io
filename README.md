# prowebleo.github.io

My corner of the internet. Three automated dashboards — price tracking, SERP monitoring, and Reddit trends — built with Next.js, libSQL, and GitHub Actions.

The portfolio itself is a single HTML file. No frameworks, no build step, no unnecessary complexity. Just a page that looks good and gets out of the way.

## What's here

- **Amazon Price Monitor** — GPU price tracking with historical charts
- **SERP Rank Tracker** — Google position monitoring for keywords
- **Trends Scout** — Reddit engagement analytics across subreddits

Built because I wanted to learn automation workflows. Kept because they're actually useful.
