My corner of the internet.

Three automated dashboards focused on price tracking, search monitoring, and online trend analysis, built with Python, Next.js, libSQL, and GitHub Actions.

The portfolio itself is intentionally simple. Fast, clean, and focused on the projects instead of unnecessary noise.

## What's here

**Amazon Price Monitor**

GPU price tracking with historical charts and automated data collection.

**SERP Rank Tracker**

Google keyword position monitoring with visual ranking insights.

**Trends Scout**

Reddit engagement analytics across subreddits to identify trending topics and activity patterns.

## Why I built this

Mostly to learn automation workflows, scraping, dashboards, and data systems.

Then the tools started becoming genuinely useful, so they stayed.
