Price Monitor

Multi-product e-commerce price tracking with trend analysis

E-commerce prices fluctuate constantly. Manually tracking multiple products across different ASINs is impractical. This system automates the process — scraping daily, detecting changes, and visualizing trends in a single dashboard.

Features

  • Automated daily scraping via scheduled jobs
  • Multi-product comparison with price overlay chart
  • Intelligent badges (Best Price, Top Rated, Most Reviews)
  • Historical price tracking with snapshot storage
  • CSV export for offline analysis

Technical Challenges

  • Handling inconsistent HTML structures across products
  • Rate limiting and retry logic for reliable scraping
  • Data normalization across different currencies and formats
  • Chart performance with growing historical datasets

Rank Tracker

Automated Google Search position monitoring for any keyword set

SEO monitoring requires daily position checks. Manual checking is slow and inconsistent. This system runs automated Google searches, extracts organic results, and tracks ranking changes over time with interactive visualizations.

Features

  • Daily automated keyword position scraping
  • Color-coded rank badges (top 3, top 10, outside)
  • Interactive rank history chart with time range selection
  • Trend indicators for position improvement or decline
  • CSV export for reporting

Technical Challenges

  • Handling dynamic search result structures
  • Parsing inconsistent result formats across queries
  • Managing rate limits for automated search requests
  • Rendering large position datasets efficiently

Trends Scout

Reddit trend intelligence — community engagement tracking

Understanding what content gains traction in online communities is valuable for marketing and product research. Manually browsing subreddits is inefficient. This system scrapes top posts daily, analyzes engagement metrics, and presents actionable trends.

Features

  • Multi-subreddit daily content scraping
  • Engagement analysis (upvotes, comments, upvote ratio)
  • Community selector with instant switching
  • Top post cards with upvote percentage bars
  • CSV export per community

Technical Challenges

  • Rate limiting across multiple subreddit requests
  • Handling Reddit API response variations
  • Normalizing post data with missing fields
  • Efficient data aggregation for engagement charts
